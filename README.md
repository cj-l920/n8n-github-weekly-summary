# n8n GitHub Weekly Activity Summary Workflow

A comprehensive n8n workflow that automatically generates weekly summaries of GitHub repository activity using AI-powered analysis.

## 🎯 Features

- **Automated Scheduling**: Runs weekly via built-in scheduler
- **Multi-Repository Support**: Can be configured for multiple repos
- **AI-Powered Summaries**: Uses OpenAI GPT-4 for intelligent activity summarization
- **Multiple Output Formats**:
  - Slack notifications
  - Email reports
  - Markdown file export
- **Comprehensive Metrics**:
  - Commits and contributors
  - Pull requests (opened, merged, closed)
  - Issues (opened, closed, labels)
  - Releases
  - Code changes (additions/deletions)

## 📋 Prerequisites

- n8n instance (self-hosted or cloud)
- GitHub Personal Access Token
- OpenAI API Key
- (Optional) Slack workspace with bot integration
- (Optional) SMTP server for email notifications

## 🚀 Installation

### 1. Import the Workflow

1. Download `workflow.json`
2. In n8n, go to **Workflows** → **Import from File**
3. Select the downloaded JSON file

### 2. Configure Credentials

#### GitHub Credentials
1. Go to **Settings** → **Credentials**
2. Create new **GitHub API** credential
3. Enter your Personal Access Token

#### OpenAI Credentials
1. Create new **OpenAI API** credential
2. Enter your API key

#### Slack Credentials (Optional)
1. Create a Slack app at https://api.slack.com/apps
2. Add `chat:write` scope
3. Install to workspace
4. Copy Bot User OAuth Token to n8n credentials

### 3. Configure Workflow Parameters

Edit the **Schedule Trigger** node to set your preferred schedule (default: weekly on Monday at 9 AM).

### 4. Set Repository Information

Modify the workflow input to specify your target repositories:

```json
{
  "owner": "your-org",
  "repo": "your-repository",
  "slackChannel": "#github-activity",
  "toEmail": "team@example.com",
  "fromEmail": "github-summary@example.com"
}
```

## 🔧 Customization

### Adjusting Time Range

Edit the **Aggregate Activity Data** node to change the reporting period:

```javascript
// Default: Last 7 days
const oneWeekAgo = new Date(now.getTime() - 7 * 24 * 60 * 60 * 1000);

// For 14 days:
const twoWeeksAgo = new Date(now.getTime() - 14 * 24 * 60 * 60 * 1000);
```

### Custom AI Prompts

Modify the **Generate AI Summary** node to customize the summary style:

```javascript
{
  "role": "system",
  "content": "Your custom instructions here..."
}
```

### Adding More Repositories

Use the **Split In Batches** node to process multiple repositories in parallel.
